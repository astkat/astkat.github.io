Alone run

Date: 二月 11, 2017  0:00:00 Category: 片刻

很早就有舍弃现在这个杂米ieza.cn的意图 但一直没有好的域名去选择 并且近来又偏好一些后缀非主流的域名  于是选了一些后敲定了这个alone.run

但主机在国内 备案现在也没有时间 就先闲置吧

现在这个域名还是会一直续费并做转向的 因为实在也不知道是否可能再会换域名

另外 可能是工单的作用或者他们自己良心发现 发觉主机管理中心现在改版的愈加人性化 主机服务也越来越进步 总之 百度云(没钱续费转到了vultr)还是很棒的选择
